🌙 Crepúsculo – A Saga de Bella & Edward
Um site interativo inspirado no universo de Crepúsculo, apresentando a história do livro, resumo dos capítulos e os principais personagens da saga.
O projeto utiliza HTML, CSS e JavaScript para criar uma experiência visual imersiva com navegação por abas, design temático e animações suaves.

📖 Sobre o Projeto
Este projeto apresenta uma página temática baseada no livro Crepúsculo, contando a história de Bella Swan e Edward Cullen.
O site inclui:
Introdução à história
Resumo dos capítulos do livro
Apresentação dos personagens principais
Interface visual inspirada na atmosfera da saga
A navegação é feita por abas interativas, permitindo alternar entre as seções sem recarregar a página.

🛠️ Tecnologias Utilizadas
HTML – Estrutura da página
CSS3 – Estilização, layout responsivo e animações
JavaScript – Controle da navegação entre abas
Google Fonts – Tipografia personalizada

🎨 Funcionalidades
✔ Navegação por abas (História, Capítulos e Personagens)
✔ Layout responsivo
✔ Animações suaves de entrada
✔ Cartões de personagens com imagens
✔ Design temático inspirado na saga
